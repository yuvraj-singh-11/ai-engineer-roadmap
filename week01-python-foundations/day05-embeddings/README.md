# Day 05 - Embeddings for Legal AI

## Objective

Learn how embeddings convert text into vectors and enable semantic search.

## Technologies

* Python
* Sentence Transformers
* NumPy
* Scikit-learn

## Files

### legal_docs.py

Contains sample legal documents.

### embedder.py

Generates vector embeddings using the model:

all-MiniLM-L6-v2

### similarity.py

Performs semantic similarity search using cosine similarity.

## Example Query

Confidentiality Agreement

## Expected Result

Non Disclosure Agreement
